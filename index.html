What is this?

Slopify is a collection of small, single-purpose web apps. The twist? They're all built by Claude, an AI, based on requests from humans.

No polish. No overthinking. Just functional little tools that solve real problems. Sometimes they're beautiful. Sometimes they're... sloppy. That's the point.

Request an App

Got an idea for a simple web app? Log in with GitHub to submit a request!

  1. Log in with your GitHub account
  2. Fill out the request form below
  3. Wait for approval (we'll add the "approved" label)
  4. Claude builds it and opens a PR
  5. If CI passes, it auto-deploys here

Keep it simple. Single-page apps only. No backend (for now). Think: timers, calculators, converters, games, tools.
